# atividade-pratica---GUUI
A atividade prática com o Gui foi desenvolvida para aplicar, de forma leve e objetiva, os conteúdos trabalhados em aula. Nela, o Gui realiza tarefas guiadas que estimulam a compreensão, a prática e a autonomia, permitindo que ele aprenda fazendo. A proposta envolve participação ativa, uso de exemplos simples e resolução prática das situações apresentadas.
